## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

This projects shows how users can gain points through their actions as posting, commenting, liking or disliking among other. Users also gain previllages based on the number of points they have using Role-Based Access Control (RBAC) as being moderators.

Here is what the project does:

- Allow users to automatically register and login.
- Allow users to CRUD posts and comments.
- Automatically allocate points based on user activity
- Automatically change user role from basic to moderator based on the number of point accumilated.

### Built With

- ReactJs
- PassportJS
- MariaDB
- Apollo GraphQL

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ensure you have the latest version of nodejs

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Cavein254/rewardsystem
   ```
2. Install the Client
   ```sh
   cd client
   npm install
   ```
3. Install the Server
   ```sh
   cd..
   cd server
   npm install
   ```
